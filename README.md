# Busca endereço pelo cep
## Funções:
- Busca endereço pelo cep de qualquer lugar do Brasil
### Apresenta:
- Lougourado (Rua)
- Cep
- Cidade e muito mais...
## Observações:
- Digite apenas numeros sem traços ou letras
